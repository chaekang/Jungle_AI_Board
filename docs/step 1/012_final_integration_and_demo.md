# 012_final_integration_and_demo

## 목적

이 문서는 [implementation_order.md](../implementation_order.md)의
`13. 최종 결론`과 그 직전의 전체 통합 흐름을 실제 점검 / 시연 관점으로 정리한 문서다.

이 단계는 기능 추가보다,
지금까지 만든 기능을 안정적으로 보여줄 수 있게 묶는 단계다.

관련 개념 문서:

- [docs/concept/012_final_integration_and_demo.md](../concept/012_final_integration_and_demo.md)

## 이 단계가 끝나면 되는 것

- 기본 게시판 기능부터 AI 기능까지 시연 흐름이 이어진다
- 주요 실패 지점을 미리 점검할 수 있다
- 데모 질문과 기대 결과가 정리된다

## 권장 점검 축

- 기능 통합
- 데이터 품질
- 에러 대응
- 시연 스크립트

## 권장 최종 시연 순서

1. 회원가입 / 로그인
2. 후기 작성
3. 댓글 / 태그 확인
4. 검색 / 필터 시연
5. RAG 질문 시연
6. MCP 기반 보조 정보 시연
7. Agent 추천 시연

## 미리 준비할 것

### 데모용 데이터

- 극장 2~3개
- 작품 2~3개
- 후기 여러 개
- 태그
- 배우 / 배역 관련 메모가 들어간 후기

### 데모용 질문

- "블루스퀘어 2층 3열 괜찮아?"
- "표정 보기 좋은 자리 추천해줘"
- "시야 방해 적고 편한 자리 위주로 골라줘"

## 최종 점검 체크리스트

- [ ] 인증 흐름 시연 가능
- [ ] 후기 작성부터 검색까지 시연 가능
- [ ] RAG 답변과 근거 후기 노출 가능
- [ ] MCP 실패 시 fallback 확인
- [ ] Agent 추천 응답 확인
- [ ] 주요 에러 상황 설명 가능

## 완료 기준

- 기능 설명이 아니라 실제 흐름으로 제품을 보여줄 수 있다
- 데모 중 막힐 가능성이 큰 지점을 사전에 줄였다
